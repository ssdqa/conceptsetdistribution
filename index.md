# Concept Set Distribution

The Concept Set Distribution is a Concept Set Testing module that
summarizes the usage of different codes within a user-provided concept
set. This allows the user to investigate, for a given study variable
definition, which specific concepts (ex: ICD10, LOINC) are driving
cohort inclusion.

## Installation

You can install the development version of this package like so:

``` r
devtools::install_github('ssdqa/conceptsetdistribution')
```
